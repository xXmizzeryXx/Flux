# Flux
A fast web proxy built for the Eclipse Suite.

## Features
- Rewrites links, scripts, and network requests so pages load fully
- Service Worker based — no extensions or installs needed
- Search anything directly from the URL bar
- Built into the Eclipse Suite aesthetic
- Free to run on Cloudflare's free tier
